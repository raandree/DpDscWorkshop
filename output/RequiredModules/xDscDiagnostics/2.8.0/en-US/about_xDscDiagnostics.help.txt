TOPIC
    about_xDscDiagnostics

SHORT DESCRIPTION
    This module helps in reading details and analyzing DSC events.

LONG DESCRIPTION
    This module contains cmdlets for analyzing DSC event logs and identifying the
    causes of any failure in a DSC operation.

EXAMPLES
    PS C:\> Get-DscResource -Module xDscDiagnostics

    writing code, sharing opinions, and provided feedback.
